# Horizon AI 新闻雷达：当前配置状态

## 已经完成

- 已从官方 GitHub 项目下载 Horizon。
- 已建立项目专用 Python 环境 `.venv`。
- 已安装基础依赖和测试工具。
- 已创建本机配置 `data/config.json`。
- 已准备 GitHub 自动运行配置 `data/config.github.json`。
- 已关闭邮件、飞书通知、Twitter、Telegram、OpenBB 等需要额外账号的功能。
- 已配置公开新闻来源：Hacker News、Reddit、GitHub、RSS、arXiv 和 Google News。
- 日报默认生成中文和英文两个版本。

## 还需要用户完成

### 1. 准备一个 AI API Key

API Key 中文可以理解为“调用 AI 服务的专用密码”。

当前配置使用 DeepSeek，环境变量名称是：

`DEEPSEEK_API_KEY`

本机使用时，把密钥填写到项目根目录的 `.env` 文件中。不要把完整密钥发到聊天中，也不要把 `.env` 上传到 GitHub。

GitHub 自动运行时，把同一个密钥添加到仓库的 GitHub Secrets 中，名称同样填写 `DEEPSEEK_API_KEY`。

### 2. 决定运行方式

- 本机手动运行：适合先测试，电脑开着时手动执行。
- GitHub Actions 自动运行：由 GitHub 每天运行，不要求你的电脑一直开机。

### 3. 根据需要调整关注内容

新闻来源和筛选规则都在 `data/config.json` 与 `data/config.github.json` 中。当前是通用 AI 新闻配置，后续可以增加股票、期货、商品、行业或指定公司的消息源。

## 重要提醒

- ChatGPT 或 Codex 会员不等于 OpenAI API Key，两者是不同的计费系统。
- 不要把密钥直接写进 `data/config.json`。
- `.env` 已被项目的 `.gitignore` 忽略，不会正常上传到 GitHub。
- 第一次运行前，应先验证配置和测试结果。
